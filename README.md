# Healthcare Analytics System

## Overview

The Healthcare Analytics System is an end-to-end data analytics project developed to analyze operational, clinical, financial, laboratory, and insurance data within a healthcare environment.

The project demonstrates the complete analytics workflow, beginning with relational database design in MySQL, followed by advanced SQL analysis, and concluding with interactive dashboards developed in Microsoft Excel, Power BI, and Tableau. The solution provides meaningful business insights to support data-driven decision-making in healthcare management.

---

## Objectives

- Design a normalized relational healthcare database.
- Perform data cleaning and validation.
- Develop advanced SQL queries for healthcare analytics.
- Build interactive dashboards using Excel, Power BI, and Tableau.
- Analyze patient, doctor, treatment, laboratory, billing, and insurance data.
- Generate business insights through data visualization.

## Database

The database consists of ten normalized relational tables:

- Patient
- Doctor
- Department
- Visit
- Treatment
- Lab Test
- Billing
- Insurance Policy
- Insurance Claims
- Insurance Provider

The database follows Third Normal Form (3NF) and maintains referential integrity using Primary Keys and Foreign Keys.
