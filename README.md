# 🏥 Healthcare Analytics System

An end-to-end Healthcare Analytics project that demonstrates the complete data analytics lifecycle using **MySQL, Microsoft Excel, Power BI, and Tableau**. This project transforms raw healthcare data into meaningful business insights through relational database design, advanced SQL analytics, and interactive business intelligence dashboards.

---

## 📖 Project Overview

The Healthcare Analytics System was developed to analyze hospital operations, patient demographics, doctor performance, treatments, laboratory diagnostics, billing, and insurance claims.

The project follows a complete analytics workflow—from database design and SQL development to interactive dashboard creation—providing meaningful insights that support data-driven decision-making in healthcare organizations.

---

## 🎯 Project Objectives

- Design a normalized relational healthcare database.
- Ensure data integrity using Primary Keys and Foreign Keys.
- Perform advanced healthcare analytics using SQL.
- Build interactive dashboards in Excel, Power BI, and Tableau.
- Analyze operational, clinical, financial, laboratory, and insurance data.
- Generate actionable business insights for healthcare management.

## 🗄️ Database Overview

The database consists of **10 normalized relational tables**.

- Patient
- Doctor
- Department
- Visit
- Treatment
- Lab Test
- Billing
- Insurance Policy
- Insurance Claims
- Insurance Provider

The database follows **Third Normal Form (3NF)** to minimize redundancy while maintaining referential integrity.
